This project presents the design and implementation of an interactive 3D district map of Bangladesh using modern OpenGL 4.1 and C++17. The system renders all 64 districts of Bangladesh across 8 administrative divisions as extruded three-dimensional polygon meshes, each coloured according to their respective division. The application incorporates spring-physics-based animation that causes each district to rise and fall independently in a fluid, organic motion. A Physically Based Rendering (PBR) lighting model — comprising directional sunlight, ambient sky illumination, specular highlights, and rim glow — gives each district a realistic, tactile appearance. An animated ocean plane using Gerstner wave vertex shaders surrounds the landmass, and water bodies. User interaction is handled through a raycasting-based hover and selection system coupled with an orbital camera, enabling full rotation, pan, and zoom. A Dear ImGui heads-up display overlaid on the 3D viewport presents district information, a division legend, and keyboard controls in real time. The application loads real geographic boundary data from GADM GeoJSON files and falls back to a comprehensive built-in dataset of simplified district polygons when no external file is present. The project demonstrates the integration of spatial data processing, real-time 3D rendering, and interactive user interface design within a single cohesive software system.
